## Configuration

| 描述   | 详情                                                  |
| ------------------- | ------------------------------------------- |
| 电脑型号      | 华为MateBook D2018     |
| 处理器           | 英特尔酷睿i5-8250U处理器     |
| 内存              | 12 GB 2400 MHz DDR4              |
| 硬盘           | SanDisk SD9SN8W256G1027 SATA    |
| 显卡 | Intel UHD Graphics 620 2048 MB                     |
| 声卡         | Realtek ALC256           |
| 无线网卡       | BCM94352Z                        |


## 正常工作的设备

- 显卡正常
- 亮度调节完美
- 无线网卡（需要拆机更换无线网卡）
- 触摸板（手势完美）
- 摄像头
- 声卡 Realtek ALC256
    - 麦克风和耳机都很正常，但是插耳机睡眠唤醒后有问题，重新插拔耳机即可
- 显示器
    - 休眠后需要重新开合才能点亮显示器
  

## 不能正常工作的设备

- 独显 MX150
    - 已经用补丁进行屏蔽
